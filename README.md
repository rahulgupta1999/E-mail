# Email Portal
#### It is local email portal where user by registering can login,compose mail,see send-box,see inbox and can change password . 
- ***Language use - Python***
- ***Framework - Django***
- ***Architecture - MVC***
- ***Database - Sqlite***
- ***For Frontend - Bootstrap,Html5,CSS***
- ***For Client Side scripting - Jquery***
